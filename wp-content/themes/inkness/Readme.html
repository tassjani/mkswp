Creative and Beautiful WordPress Theme

Description

Inkness is a unique, ultra creative multipurpose WordPress theme with complete support for WooCommerce. Inkness supports all basic features and is fully mobile friendly. And with Support for Custom Widgets, Slider and Featured Posts section, etc you can make this theme look even more beautiful on your website. Demo Here: http://demo.inkhive.com/inkness/
